# Cure App – Overview

Diese Datei beschreibt den Gesamtzweck der Cure-App.

Kurz: Die Cure-App ist eine Flutter-Anwendung zur Kommunikation mit den CureBase / Pyramide Geräten über BLE. Sie bietet Nutzerführung zum Verbinden, Hochladen von Programmen, Start/Stop von Programmen und Telemetrie.

Ziele:
- Native Apps für Android und iOS (Flutter)
- Stabile BLE-Kommunikation (Nordic UART Service)
- Sichere Unlock-/Authentifizierungs-Handshake
